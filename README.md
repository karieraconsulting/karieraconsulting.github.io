# Kariera Advisory

> Landing page institucional da Kariera, especializada em Advisory estratégico e aceleração para Consultorias de Tecnologia e TI.

## 🌐 Link de Produção
O site está publicado e disponível em: **[https://karieraconsulting.vercel.app/](https://karieraconsulting.vercel.app/)**

## 🛠️ Tecnologias Utilizadas
- **HTML5** para a estrutura semântica.
- **Tailwind CSS (via CDN)** para estilização rápida e responsiva.
- **JavaScript (Vanilla)** para a rolagem suave.
- **Google Fonts (Inter)** para tipografia moderna.

## 🚀 Como Executar Localmente
Para visualizar o projeto localmente, basta abrir o arquivo `index.html` diretamente no seu navegador ou utilizar uma extensão como o *Live Server* no seu editor de código.

## 📦 Deploy na Vercel
O deploy do projeto é gerenciado através da Vercel CLI. Para realizar novos deploys de produção diretamente do seu terminal, execute:
```bash
npx vercel --prod --scope felipersb-3663s-projects
```
